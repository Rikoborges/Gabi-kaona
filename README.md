# Gabi Kaoana — Confeitaria Artesanal

Site institucional com blog para a Gabi Kaoana, confeitaria artesanal de Curitiba especializada em brigadeiros, brownies e pipoca gourmet.

## 🔗 Deploy

[gabikaoana.vercel.app](https://gabikaoana.vercel.app)

## 📁 Estrutura
```
gabi-kaoana/
├── index.html
├── politica-privacidade.html
├── assets/
│   ├── css/style.css
│   ├── js/main.js
│   └── images/
└── blog/
    ├── index.html
    ├── historia-do-brigadeiro.html
    ├── segredo-brigadeiro-perfeito.html
    ├── bastidores-gabi.html
    └── tipos-chocolate.html
```

## ✅ Funcionalidades

- Landing page com cardápio completo e preços
- Blog com 4 posts e filtros por categoria
- Página LGPD completa (Privacidade, Termos, Cookies)
- Banner de cookies funcional
- WhatsApp flutuante
- Botão voltar ao topo
- Totalmente responsivo — mobile first

## 🛠 Tecnologias

- HTML5 semântico
- CSS3 — custom properties, grid, scroll-driven animations
- JavaScript vanilla
- Schema.org JSON-LD (FoodEstablishment + FAQPage)
- SEO técnico completo — geo tags, Open Graph, meta tags
